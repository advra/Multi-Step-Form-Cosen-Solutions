## Getting Started

Multistep from based off of [Cosden Solutions Youtube Video](https://www.youtube.com/watch?v=vKJe95pqn8E)

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Navigate to: http://localhost:3000/onboarding/name
